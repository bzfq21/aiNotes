# AI Notes 🧠

个人AI笔记仓库，系统化记录人工智能领域的知识点、项目经验和思考总结

## 📖 项目背景
在人工智能领域，知识体系庞大且更新迅速。为了更好地理解和应用这些知识，我决定创建一个个人笔记仓库，系统化地记录学习过程中的知识点、项目经验和思考总结。这个仓库旨在帮助我巩固知识，同时也为其他学习AI的朋友提供参考。

## 🚀 快速开始

1. **克隆仓库**
   ```bash
   git clone <this-repo-url>
   cd aiNotes
   ```

2. **查看目录结构**
   ```bash
   tree -I '.git'
   ```

3. **开始记录**
   - 查看 `templates/` 目录下的模板文件
   - 根据学习内容选择合适的模板
   - 在对应目录下创建新的笔记文件

## 📝 笔记规范

### 文件命名
- 使用英文，小写字母和连字符：`transformer-architecture.md`
- 论文笔记：`[年份]-[论文简称].md` 例如：`2027-attention.md`
- 项目记录：`[项目名]-[日期].md`

### 内容结构
每个笔记应包含：
- **概述**：简要介绍主题
- **核心概念**：关键知识点解释
- **实践应用**：代码示例或实际应用
- **思考总结**：个人理解和疑问
- **参考资料**：相关链接和引用

## 🛠️ 推荐工具

## 📚 学习路径建议

1. **基础阶段** → `docs/fundamentals/`
   - 数学基础（线性代数、概率论、微积分）
   - 编程基础（Python、数据结构）

2. **核心算法** → `docs/machine-learning/`
   - 监督学习、无监督学习
   - 特征工程、模型评估

3. **深度学习** → `docs/deep-learning/`
   - 神经网络基础
   - CNN、RNN、Transformer

4. **专业方向** → 根据兴趣选择
   - NLP、CV、强化学习等
