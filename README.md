# 目录说明

这个目录包含了AI学习笔记的核心内容，按主题分类组织。

## 📂 子目录说明

- **fundamentals/** - AI基础理论（数学、统计、算法基础）
- **machine-learning/** - 传统机器学习算法和方法
- **deep-learning/** - 深度学习相关内容
- **nlp/** - 自然语言处理
- **cv/** - 计算机视觉
- **reinforcement/** - 强化学习
- **tools/** - AI工具和框架使用指南

## 📝 写作建议

1. **理论结合实践** - 每个概念尽量配上代码示例
2. **循序渐进** - 从基础到高级，构建知识体系
3. **定期回顾** - 定期复习和更新笔记内容
